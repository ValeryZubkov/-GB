# Инструкция по работе с Git

## Что такое Git

Git - это одна из реализаций распределенных систем контроля версий, что позволяет иметь версионность как в локальном репозитории, так и в удаленном репозитории (общем для всех). Git является на данный момент самой популярной системой контроля версий.

## Подготовка репозитория

Для создания репозитория используется команда *git init*

## Перемещение между коммитами

Для того, чтобы перемещаться между коммитами необходимо использовать команду "git checkout". Для этого в иерминале с папкой репозитория необходимо написать "git checkout <номер коммита>"...
